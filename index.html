Recent Work

House Marketplace

This is a House Marketplace project developed in React. People can register and list their home for rent or sale. Users also can see other's listing and contact the landlord. The User authentication for sign in, sign up and the listing is done by Google Firebase. All the house images and house information is also stored in Firebase. I used 'react-tostify' to show the error messages and alerts.

Meal Finder

This is a Meal Finder developed in React and SASS. Meal Finder uses TheMealDB API(https://www.themealdb.com/api.php). Uers can search recipes by entering any key terms such as ingredient, way of cooking, category, etc. Detailed ingredients and directions will show up by clicking any recipe cards searched.

Jewelry Store Shopping Cart

This is a Jewelry store shopping cart project. The shopping cart is implemented by using only Vanilla JavaScript. Users can add items into the cart, increase or decrease the quantity by clicking the up and down arrows. They also can remove individual items from the shopping cart or remove all the items by clicking 'Clear Cart' button. When some items are already in the cart, hover buttons show 'In Cart' labels. Product data is stored in both JSON file and the local storage. The cart information is also saved in the local storage to keep the items saved even after the reload.
